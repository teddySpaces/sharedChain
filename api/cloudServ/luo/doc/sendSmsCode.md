# sendSmsCode
    发送手机验证码
    
### Url
    /capp/users/sendsms
    
### Method
    POST

### Request Payload
    {
        "mobile":"手机号码"
    }
    
### Response Body
    {"status":"OK"}
    
### Response Code
    HTTP/1.1 200 OK

### Example
    curl -X POST  "http://127.0.0.1:8065/capp/users/sendsms"  -d '{"mobile":"13544285662"}'  -D cookie-file -i
