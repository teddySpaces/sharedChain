# isMobileExist
    判断手机号是否已经注册过
    
### Url
    /capp/users/phone/exist
    
### Method
    POST

### Request Payload
    {
        "mobile":"13544285663"
    }

### Response Body
    {
        "status":"true"
    }
    
### Response Code
    HTTP/1.1 200 OK

### Example(存在)
    curl -X POST  "http://127.0.0.1:8065/capp/users/phone/exist"  -i -d '{"mobile":"13544285663"}'
