# resetPasswordByMobile 
    手机号找回密码

### Url
    /capp/users/phone/reset

### Method
    POST

### Request Payload
    {
        "mobile":"手机号码",
        "verification_code":"验证码",
        "new_password":"新密码"
    }


### Response Body
    {
        "status":"OK"
    }

### Response Code
    HTTP/1.1 200 OK


### Example
	curl -X POST  "http://127.0.0.1:8065/capp/users/phone/reset"  -i -d '{"mobile":"13544285662","verification_code":"666666","new_password":"4538***************1"}'
