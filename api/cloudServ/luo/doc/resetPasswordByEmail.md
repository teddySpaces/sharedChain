# resetPasswordByEmail 
    邮箱找回密码

### Url
    /capp/users/email/reset

### Method
    POST

### Request Payload
    {
        "email":"邮箱地址",
        "verification_code": "",
        "new_password":""        
    }


### Response Body
    {
        "status":"OK"
    }


### Response Code
    HTTP/1.1 200 OK


### Example
    curl -X POST http://www.example.com:8866/capp/users/email/reset -d '{"email":"1027837952@qq.com","verification_code":"733644","new_password":"Zhang12345678"}' -i
